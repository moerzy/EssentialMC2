## 安装

### 依赖项
- Python 3.6+
- PyTorch 1.5.1+
- torchvision 0.6.1+

### 安装EssentialMC2
#### 安装稳定版本
```shell
pip install esscm2
```

#### 安装开发版本

首先克隆最新版本的EssentialMC2至本地
```shell
git clone https://github.com/alibaba/EssentialMC2.git
cd EssentialMC2
```

然后执行安装命令
```shell
python setup.py develop
```

注意，在完成安装开发版本后，任何对EssentialMC2下的代码修改都会直接生效。
